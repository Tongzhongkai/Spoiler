Possible colliding addresses are marked with a "###"
value between "()" is the distance to the last colliding address

Make sure that SharedArrayBuffers are enabled (enabled by default in chrome >=68).

How to enable SharedArrayBuffers?
firefox:
about:config => javascript.options.shared_memory => true

chrome, chrome based browser (e.g. opera):
chrome://flags/ => shared-array-buffer => true
